🔒 Política de Privacidade

Roku Streaming Server - Transparência e Proteção de Dados

📋 Resumo Executivo: O Roku Streaming Server é projetado com privacidade em mente. Não coletamos dados pessoais dos usuários, não utilizamos cookies de rastreamento e não compartilhamos informações com terceiros. Esta aplicação funciona como um servidor de streaming local/auto-hospedado.

1. 🌐 Visão Geral

Esta Política de Privacidade descreve como o Roku Streaming Server trata as informações quando você utiliza nossa aplicação de streaming. Nosso compromisso é com a transparência e a proteção da sua privacidade.

Informações importantes:

  • ✅ Não coletamos dados pessoais dos usuários finais
  • ✅ Não utilizamos cookies de rastreamento ou analytics
  • ✅ Não compartilhamos dados com terceiros
  • ✅ Código open source - você pode auditar nossa implementação
  • ✅ Auto-hospedado - você controla seus próprios dados

2. 📊 Coleta de Dados

2.1 Dados que NÃO coletamos:

  • ❌ Informações pessoais identificáveis (nome, e-mail, telefone)
  • ❌ Dados de localização geográfica
  • ❌ Histórico de visualização detalhado
  • ❌ Informações de dispositivo pessoal
  • ❌ Dados biométricos ou comportamentais
  • ❌ Informações financeiras

2.2 Dados técnicos mínimos (logs do servidor):

  • 🔧 Logs de acesso: Endereços IP, timestamps, endpoints acessados
  • 🔧 Logs de erro: Informações técnicas para debugging
  • 🔧 Métricas de performance: Tempo de resposta, uso de recursos
💡 Importante: Estes logs são armazenados localmente no servidor e são utilizados exclusivamente para manutenção técnica e resolução de problemas. Não contêm informações pessoais identificáveis.

3. 🎯 Uso dos Dados

Os dados técnicos mínimos coletados são utilizados exclusivamente para:

  • 🛠️ Manutenção técnica: Identificar e corrigir problemas no servidor
  • 📊 Monitoramento de performance: Otimizar a velocidade e estabilidade
  • 🔒 Segurança: Detectar tentativas de acesso malicioso
  • 📈 Melhorias: Entender padrões de uso para otimizações
✅ Garantia: Nunca utilizamos dados para fins comerciais, publicitários ou de marketing. Não criamos perfis de usuário nem fazemos análises comportamentais.

4. 🤝 Compartilhamento de Dados

Política de não compartilhamento:

  • ❌ Não vendemos dados para terceiros
  • ❌ Não compartilhamos com parceiros comerciais
  • ❌ Não fornecemos para fins publicitários
  • ❌ Não integramos com redes sociais para coleta de dados

Exceções legais:

Podemos divulgar informações apenas se:

  • ⚖️ Exigido por lei ou ordem judicial
  • 🛡️ Necessário para proteger direitos legais
  • 🚨 Prevenir fraude ou atividade ilegal

5. 💾 Armazenamento de Dados

5.1 Localização:

  • 🏠 Auto-hospedado: Dados ficam no seu próprio servidor
  • 🌐 Deploy em nuvem: Dados ficam no provedor escolhido (Render, Heroku, etc.)
  • 🔒 Sem transferência internacional: Dados não saem da região configurada

5.2 Retenção:

  • 📅 Logs de acesso: 30 dias (configurável)
  • 🐛 Logs de erro: 90 dias (para debugging)
  • 📊 Métricas: 7 dias (apenas para monitoramento)

5.3 Segurança:

  • 🔐 HTTPS obrigatório em produção
  • 🛡️ Headers de segurança configurados
  • 🔒 Acesso restrito aos logs do servidor

6. 👤 Direitos do Usuário

Como não coletamos dados pessoais, a maioria dos direitos de privacidade não se aplica. No entanto, você tem:

  • 🔍 Direito à transparência: Esta política e código open source
  • 📋 Direito à informação: Saber exatamente o que fazemos
  • 🚫 Direito de não uso: Parar de usar o serviço a qualquer momento
  • 🛠️ Direito de controle: Hospedar sua própria instância
💡 Auto-hospedagem: Para controle total sobre seus dados, recomendamos hospedar sua própria instância do Roku Streaming Server.

7. 🔒 Segurança

Implementamos as seguintes medidas de segurança:

7.1 Segurança Técnica:

  • 🔐 HTTPS/TLS: Criptografia em trânsito
  • 🛡️ Headers de segurança: HSTS, CSP, X-Frame-Options
  • 🚫 CORS configurado: Controle de acesso de origem
  • ⚡ Rate limiting: Proteção contra ataques DDoS

7.2 Segurança Operacional:

  • 📊 Monitoramento: Logs de segurança e alertas
  • 🔄 Atualizações: Patches de segurança regulares
  • 🧪 Testes: Auditorias de segurança periódicas

8. 🍪 Cookies e Tecnologias Similares

✅ Boa notícia: O Roku Streaming Server NÃO utiliza cookies de rastreamento, analytics ou publicidade.

Cookies técnicos mínimos (se aplicável):

  • ⚙️ Cookies de sessão: Apenas para funcionalidade básica
  • 🎛️ Preferências locais: Configurações do player de vídeo
  • 🔒 Cookies de segurança: Proteção CSRF (se implementado)

Não utilizamos:

  • ❌ Google Analytics ou similares
  • ❌ Pixels de rastreamento
  • ❌ Cookies de terceiros
  • ❌ Fingerprinting de dispositivo

9. 🔗 Serviços de Terceiros

9.1 Vídeos de demonstração:

Utilizamos vídeos de exemplo do Google Cloud Storage para demonstração. Estes vídeos são públicos e não coletam dados dos usuários.

9.2 Provedores de hospedagem:

Se você usar nossa aplicação em provedores como Render, Heroku ou Railway, suas políticas de privacidade também se aplicam à infraestrutura.

9.3 CDNs e recursos externos:

  • 📹 Vídeos: Google Cloud Storage (apenas para demos)
  • 🎨 Fontes: Fontes do sistema (sem CDNs externos)
  • 📚 Bibliotecas: Apenas dependências essenciais do Node.js

10. 👶 Proteção de Menores

O Roku Streaming Server é adequado para todas as idades:

  • ✅ Não coleta dados de menores de idade
  • ✅ Conteúdo apropriado: Vídeos de demonstração são family-friendly
  • ✅ Sem publicidade: Não há anúncios direcionados
  • ✅ Controle parental: Administradores podem controlar o conteúdo
📋 COPPA/LGPD: Como não coletamos dados pessoais, estamos em conformidade com regulamentações de proteção de menores.
